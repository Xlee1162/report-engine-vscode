# HOW TO ADD A NEW REPORT IN 1 HOUR

## 0–10 phút: Xác định report

* Mục tiêu
* Người nhận
* Tần suất

---

## 10–25 phút: SQL

* Viết / chỉnh stored procedure
* Trả về dataset đã tính toán xong

---

## 25–35 phút: Excel (optional)

* Tạo / chỉnh template
* Thêm bảng / chart cần thiết

---

## 35–45 phút: Report Config

* Khai báo schedule
* Khai báo data scope
* Khai báo render blocks

---

## 45–55 phút: Test render

* Chạy engine local
* Check mail body
* Check attachment

---

## 55–60 phút: Enable

* Bật scheduler
* Theo dõi log

---

## Kết luận

> Nếu report mới cần sửa engine → kiến trúc đã sai.
> Thêm report **chỉ là thêm config + SQL**.
